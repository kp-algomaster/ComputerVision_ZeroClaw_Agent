# CV_Research

Computer Vision Research Knowledge Vault — powered by CV Zero Claw Agent.

Open this folder in [Obsidian](https://obsidian.md) to explore the knowledge graph visually.

## Structure

- **papers/** — Individual paper notes with metadata and wiki-links
- **methods/** — Technique and method notes (architectures, loss functions, etc.)
- **datasets/** — Dataset descriptions and benchmarks
- **tasks/** — CV task definitions (detection, segmentation, etc.)
- **authors/** — Research group and author profiles
- **MOCs/** — Maps of Content — topic hub notes
- **daily/** — Daily research log entries

## How It Works

The CV agent automatically creates and links notes when processing papers:

1. Paper is fetched from ArXiv
2. Key information is extracted (methods, datasets, equations)
3. An Obsidian note is created with YAML frontmatter and `[[wiki-links]]`
4. Related entities are linked in the graph
5. MOCs are updated for relevant topics

Open Obsidian Graph View to see the connections between papers, methods, and datasets.
